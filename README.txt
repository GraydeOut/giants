Instructions:
*************
To run this project start from the Main class. Follow prompts in the console to enter time ranges for 
arrival and service durations. Then enter the number of customers to be simulated. Note that any time 
and customers must be greater than zero. After providing the initialconditions the program will simulate 
the queues and then output to console some statistics.

High Line usage Starting Values: Arrival 1-3, Service 3-7 
High Customer Satisfaction Starting Values: Arrival 1-3, Service 3-6
